# Learnova 📊

Learnova is a collection of Data Analytics and Machine Learning projects focused on Exploratory Data Analysis (EDA), Data Visualization, and Predictive Modeling using Python.

This repository contains real-world datasets and projects built using libraries like Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sagar2508/learnova.git
```

Move into the project directory:

```bash
cd learnova
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# ▶️ Running the Projects

Run Python files:

```bash
python filename.py
```

Open Jupyter notebooks:

```bash
jupyter notebook
```
